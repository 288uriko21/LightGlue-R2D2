# Image Matching with Neural Networks

Код к выпускной квалификационной работе «Методы сопоставления изображений, снятых с разных ракурсов, с использованием нейронных сетей».

## Задача

Исследование и сравнение методов сопоставления изображений, снятых c разных ракурсов по ключевым точкам:

- выделение и описание ключевых точек;
- сопоставление ключевых точек между двумя изображениями;
- оценка качества (точность соответствий, устойчивость к изменению ракурса, скорость).

Конечная цель — подобрать и настроить связку моделей, которая даёт хороший баланс между качеством сопоставления и временем работы.

## Используемые методы

В работе рассматриваются:

- классические подходы: SIFT, ORB
- нейросетевые детекторы и дескрипторы: R2D2, SuperPoint, DISK
- нейросетевой матчинг точек: LightGlue
